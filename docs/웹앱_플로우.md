# 강화유니버스 웹앱 — 핵심 플로우

## 플로우 1 — 여행객 첫 카드 발급 (가게에서)

```
가게 방문
  → 카운터/벽의 QR 스캔 (또는 NFC 태그)
  → 웹앱 진입 (PWA, 설치 X)
  → 첫 방문이면: 휴대폰 번호 입력 + 닉네임
  → 카드 생성 화면
       ├ 사진 1장 (가게 배경/내부, 카메라 즉석 촬영)
       ├ 한 줄 메모 ("오늘 ___와 ___를 나눴다" 양식)
       └ 공개 설정 토글 (기본값: 비공개)
            ├ 비공개 → 나와 사장님만 봄
            └ 공개   → 가게 둘러보기 페이지에 노출 (홍보 자산)
  → 저장
  → "이 카드는 사장님 ___ 님께 전달됩니다" 안내
  → 본인 도감으로 자동 이동
```

**규칙**
- 사진/한 줄 둘 다 없으면 저장 불가 (게임 4규칙 ④)
- QR 스캔 없이는 카드 생성 화면 진입 불가 (오프라인 트리거 원칙)
- 같은 가게 재방문 시 → 같은 카드의 "뒷면 추가" (1관계 1카드 원칙)
- **공개 설정은 항상 기본값 비공개** — 공개는 명시적 opt-in (게임 4규칙 ③ 변형)
- 닉네임만 노출, 휴대폰 번호/실명 노출 X

## 플로우 2 — 여행객 도감 회고 (떠난 후)

```
홈 → 내 도감
  ├ 다녀온 가게 카드 리스트 (사진 그리드)
  ├ 카드 클릭 → 사진 + 한 줄 + 방문 날짜 + 공개 토글
  │      └ 언제든 공개 ↔ 비공개 전환 가능
  └ 누적 통계 (가게 N곳, 카드 N장, 사장님과 함께한 시간)

수신함 (편지 도착 시 푸시 또는 SMS)
  → 사장님이 보낸 편지 열람
  → 답장 작성 가능 (LLM 보조 + 사람 손길)
```

**규칙**
- 도감 본체는 본인 + 해당 사장님만 봄 (비공개가 기본)
- 공개로 전환한 카드만 가게 둘러보기 페이지에 노출
- 거래/양도 불가 (게임 4규칙 ①)
- 본인 도감 자체에는 검색/탐색 기능 X — 회고용 공간

## 플로우 3 — 가게 둘러보기 (외부 공개 페이지)

```
홈 또는 "강화 둘러보기" 진입
  → 가게 리스트 (지도 또는 가나다순, 랭킹 X)
       └ 가게 카드 (썸네일 + 이름 + 누적 카드 수)
  → 가게 클릭
  → 가게 페이지 (공개 뷰)
       ├ 가게 소개 (사장님이 작성, 짧은 한 줄)
       ├ 위치 / 영업시간
       └ 공개 카드 갤러리 (시간순)
            ├ 사진 + 한 줄 + 닉네임 + 방문 월
            └ 카드 클릭 시 크게 보기만, 댓글/좋아요 X
```

**규칙**
- 비로그인도 열람 가능 (홍보 효과의 핵심)
- **사장님이 가게 단위로 공개 OFF 가능** — 사장님이 거부하면 그 가게 페이지 자체가 닫힘
- 공개로 노출되는 카드는 **여행객이 공개로 토글한 것만**
- **좋아요 / 팔로우 / 댓글 / 조회수 / 랭킹 / 검색 — 전부 없음** (SNS화 방지)
- 정렬은 시간순만 (인기순 X)
- 가게 리스트도 인기 가게 X, 추천 X — 순수 인덱스
- 휴대폰/실명 노출 X, 닉네임만

## 플로우 4 — 사장님 가게 페이지 & 편지 쓰기

```
사장님: 청풍이 준 가게 코드로 로그인
  → 우리 가게 (사장님 뷰)
       ├ 다녀간 여행객 카드 모음 (공개+비공개 모두, 시간순)
       ├ 카드 클릭 → 사진 + 한 줄 + 닉네임 + 방문 날짜
       ├ 카드별로 공개/비공개 표시 아이콘
       ├ "편지 쓰기" 버튼
       └ 가게 공개 설정
            ├ 가게 소개 / 영업시간 편집
            └ 가게 페이지 공개 ON/OFF (기본값: ON)

편지 쓰기 화면
  → 카드 1장 선택
  → LLM이 카드 단서로 첫 문장 제안 ("___님, 그날 ___와 ___ 나눠주셔서…")
  → 사장님이 손으로 다듬기
  → 발송 시점 선택 (즉시 / N주 후 / N개월 후)
  → 발송 예약
```

**규칙**
- 사장님은 본인 가게 카드만 봄 (다른 가게 사장님 뷰 접근 불가)
- 사장님은 비공개 카드도 볼 수 있음 (편지 단서로 필요)
- 가게 페이지 공개를 OFF 하면 → 그 가게의 모든 공개 카드도 외부에서 안 보임
- 편지는 카드 단서 기반으로만 쓸 수 있음 (백지 X → 부담 낮춤)
- 시차 발송으로 "잊을 만할 때 도착" 효과

## 플로우 5 — 운영자 관리자 뷰

```
청풍 운영자 로그인
  → 대시보드
       ├ 누적 카드 수, 가게별 카드 수
       ├ 공개 카드 수 / 비공개 카드 수 비율
       ├ 누적 편지 수 (발신/수신/예약)
       ├ 재방문 여행객 수 (2회 이상 방문)
       ├ 활성 사장님 수 (편지 쓴 사장님)
       ├ 가게 페이지 조회 수 (홍보 효과 지표)
       └ 월별 그래프 (대표님 보고서용)
  → 가게 관리
       ├ 가게 등록 / QR 코드 발급
       ├ 사장님 가게 코드 발급
       └ 가게 페이지 공개 상태 모니터링
  → 콘텐츠 모더레이션
       ├ 신고된 사진/문구 검토
       └ 부적절 콘텐츠 비공개 전환
```

**대표님 "보이지 않는 성과" 직접 해결 지점.**
**공개 카드 비율과 가게 페이지 조회 수가 "홍보 효과"의 정량 지표가 됨.**
